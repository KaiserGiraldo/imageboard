名前

トリップコード

トリップコードを使用するには、名前に＃を追加します。 ＃の後に、トリップコードのパスワードを追加します。

サーバーはパスワードにハッシュを適用し、ハッシュされたパスワードを表示します。

安全なトリップコードを使用するには、名前とパスワードに##を追加します。 安全なトリップコードは、パスワードが一致する場合にのみ使用されるランダムなテキストを生成して表示します。

役割の署名

投稿時にグローバルまたは取締役会の役割を表示する場合は、大文字と小文字を区別せずに、トリップコードのように名前に#rsを含めることができます。 これにより、トリップコードを使用できなくなります。

アカウント設定で常に使用するロール署名ルールを有効にすることもできます。

マークダウン後

投稿するときは、HTMLのテキストを拡張するいくつかのツールを使用できます。

URL

httpまたはhttpsで始まるURLを投稿すると、自動的にリンクされます。


例：

http://bot.net → http://bot.net


緑のテキスト

追加 > 行の先頭で、改行までテキストをグリーンテキストにします。


例：

>こんにちは :^) → >こんにちは :^)


Quote

>> と書くと 次に、投稿に番号を付けると、投稿しているボードにこの投稿へのリンクが作成され、JavaScriptが有効になっている場合は、投稿のプレビューにツールチップが追加されます。


例：

>>50 → >>50


ボードリンク

>>> と書くと 次に、スラッシュの間にボード名を追加すると、ボードへのリンクが作成されます。


例：

>>>/映画/ → >>>/映画/


クロスクォート

別のボードからの投稿を引用することも可能です。 ボード名の後に投稿番号を追加するだけです。


例：

>>>/映画/50 → >>>/映画/50


サイコロを振る

すべて「m」辺を持つ「ロールアンドダイス」にするには、コマンドを使用します /roll{ndm[+-k]}
サイコロの目の上にカーソルを合わせると、修飾子付きの合計ではなく、個々のサイコロの目が表示されます。


例：

/roll{2d6} → 2d6 = 5
/roll{1d20+4} → 1d20+4 = 19
